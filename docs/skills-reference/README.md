# 技能参考文档

## 自定义技能 (`.kilo/skills/custom/`)

### apparel-collab-planning
服装行业项目节点与跨组织协作数据层规划。支持季节性项目隔离、快反供应链时效管理和多组织协同时效追踪。

**触发条件**: "规划服装行业项目数据层"、"季节性项目"、"快反供应链"、"管理跨组织协作关系"

### apparel-industry-data-planning
面向服装行业的多维表格数据层规划技能，设计供应链、生产、销售全链路数据结构。

**触发条件**: "规划服装行业数据层"、"设计供应链数据结构"、"产业链上下游映射"

### apparel-knowledge-planning
服装行业知识文档数据层规划。实现专业术语统一、跨组织文档流转和隐性知识显性化。

**触发条件**: "规划服装行业知识文档数据层"、"术语库"、"知识沉淀"、"专家经验"

### intelligent-table-data-planning
智能规划多维表格数据结构，设计主数据模型与跨表映射关系。

**触发条件**: "规划数据层"、"设计表结构"、"配置主数据"

### keyword-driven-info-processor
关键词驱动的智能信息分类与飞书表格自动更新。

**触发条件**: "整理信息"、"分类数据"、"关键词"、"写入飞书表格"

### quick-response-data-planning
快反模式人-周-项目三维度数据层规划。支持M1-M7模块结构定义、时序数据管理和RGB预警。

**触发条件**: "规划快反供应链数据层"、"M1-M7模块"、"RGB状态机"

### feishu-bitable-terminology-generator
飞书术语表合规校验方案。AI推荐替换词100%符合法规要求，三层防护+动态更新。

**触发条件**: "校验文案合规性"、"违规词"、"替换词推荐"
